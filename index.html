Puzl Time

Puzzle Game for iOS

View project on GitHub

Prepared for a real puzzle challenge? Solve over 130 challenging, time-limited sliding tile puzzles with the help of power-ups and the preview button!

In addition to the normal, time-limited sliding tile puzzles, the following challenging modes are available:

  • Hidden: Hidden preview and portrait, can you guess the image?

  • Reduced Time: Normal time left is reduced, solve as fast as possible!

  • Steps: Moves are limited to a specific number, so make sure you plan your moves well

  • Unknown: This mode is only available for the final level, if you can unlock it, or better yet... complete it. Not for the faint of heart.

Power-Ups Available:

  • Time Extend: Extends time left by 25 seconds

  • Restart Puzzle: Restarts the puzzle, with the possibility of getting an easier set-up

  • Power-ups available for use during each course (also called the "Power-up's limit") can be increased; make sure you bought enough power-ups available to satisfy your current limit.

  • Time Extend - 125 count limit, Restart Puzzle - 100 count limit

Game Center Support:

  • Challenge all Puzl Time players in the global "Highest Scores" leaderboard
  • Get the "Puzl Genius" achievement and get your score admitted to the exclusive "Puzl Geniuses" leaderboard!

*Coins can be used to unlock courses, buy power-ups and increase the limit of each power-up. The higher the score, the more coins earned. Enjoy the challenge!
